掌机屏幕取景 — 大宠不再溢出

痛点：宠物越来越大，但模拟掌机屏幕还是这么大，会不会超出范围？

现状（代码）：.stage-wrap 高 220rpx、.sprite 208rpx 方框，体型 sizeScale 由 weight 线性映射到 [0.92, 1.32]（WEIGHT_SIZE_RANGE），weight 成年上限 400。成年满膘 ≈ 208×1.32 ≈ 275rpx，已超出 220rpx 的取景高度 → 会顶出/被裁切。

现状：成年体顶破安全框 ⚠️
→
V2：永远落在安全框内 ✅

三条一起上

  • 改 取景框固定 + 安全区：把舞台抽象成一块固定 LCD（如 300rpx 高、内缩 8% 安全区）。宠物锚点 center bottom，任何形态都必须画进安全区。
  • 改 每阶段归一化基准缩放：每个 stage 的 idle 精灵渲染前先按"该 stage 目标占框比"归一（egg≈55% / baby≈62% / child≈70% / teen≈78% / adult≈84%）。"长大"的体感由换 stage 精灵 + 取景缩小表达，而不是无上限放大像素。
  • 改 收紧 within-stage 体型 band：weight 在同一 stage 内只微调 ±7%（[0.96,1.06]），保留"养胖了一点"的可见反馈，但绝不溢出。配一个相机 轻微 zoom-out：体型越大，舞台整体缩 2–4%，越显"它在画面里更有分量"。
📐 验收：adult + 满 weight + 戴最高的帽子（含 V2 新增的"光环/背景"槽）时，精灵 + 配饰 + 顶部 Zzz/心情 FX 全部落在安全区内，且 child→adult 肉眼能感到"它真的变大了一圈"。

经验曲线 — 不再"一登就升两级"

痛点：升级经验没有曲线吗？感觉每次登录都能升一两级。

现状：等级是从 exp 实时派生（不存库）：Lv = 1 + ⌊(−60 + √(3600+40·exp))/20⌋，即 到达 Lv N 累计 = 60(N−1)+10(N−1)²。低级段每级间隔只有 60–230 exp，而一个活跃日产出 ~300–450 exp（头部用户 6 天到 Lv14 ≈ 430/天），所以早期"一登升两级"，且没有每日上限去节流——成长全靠堆点击。

两个杠杆

  • 新 每日成长 EXP 软上限 dailyCap(Lv) = 300 + 12·Lv：每个本地日（午夜重置）能计入升级的成长 exp 有上限。超出后当天的照顾仍给亲密度/心情/体型反馈，但 exp 转成"明天再算"。→ 杜绝囤一波点爆 3–4 级，把成长摊成每天一点的仪式。
  • 改 更陡的每级间隔 gap(Lv) = 90 + 20·Lv + 1.1·Lv²：早期仍 ~1 级/天（有正反馈），Lv12 起每级 >1 天，Lv20+ 每级 2–3 天，Lv25+ 约 3 天/级。到 Lv30 累计 ≈ 20,700 exp（现行只要 10,150）。

濒危召回 — 吓一跳，但永不真死

痛点：可以增加宠物濒死系统吸引用户返回。
⚠️ 红线：产品核心承诺是"永不死亡、永不丢失、软可逆"。所以做的是"濒危假警报"，不是真死亡——制造担心与回访动机，但回来一摸就满血复活，绝不惩罚。

配饰系统 — 多品类 + 等级解锁

痛点：配饰系统需要增加对不同等级的限制和品类。

现状：单槽位（hat），7 顶帽子，compute-on-read 解锁（always/stage/streak/days/bond）；其中只有巫师帽(bond700)/皇冠(成年)在后期解锁，~day7 基本拿全。

🧣 颈饰 Collar围巾/铃铛/项链，Lv 解锁
🖼️ 背景 Backdrop窝/书房/海边/星空，stage+Lv

🟫 足垫 Base坐垫/草地/浮冰，days 解锁
  • 新 新增解锁类型 { type:"level"; lv:N }，把等级直接变成配饰货币——这正是"Lv 有什么用"的答案：每升几级解锁一件新装扮。
  • 解锁曲线铺满 50 天：Lv5/8/12/16/20/24/28/30 各解锁不同品类的件，配合 streak/bond/season，让"今天差 1 件就齐了"的收集欲长期在线。

成年后的"纪元" — Lv20 之后为什么还上号

痛点：玩到 20 级以后缺乏登录动力，宠物升级对我没区别，没有上号的动力。

把"上号理由"从裸等级数字解耦到一组长尾收集与情感轨道，每周都有新东西熟成：

📖 心声相册 / 记忆

  • 里程碑触发的专属心声、第一次濒危复活、每个抉择日，自动收进一本相册，按周/月成页。
  • "它还记得我们一起…"——这是最便宜的 aliveness 保险，给硬核玩家"翻看我们的历史"的理由。

🌳 星图 / 成长树

  • 一张 50 天慢慢点亮的星图：每个 stage、抉择、连登里程碑、配饰收集点亮一颗星，可视化"我们走了多远"。

🏅 称号系统

  • Lv / 天数 / bond / 收集度 解锁称号（如"相伴百日""本命满羁绊""四季见证者"），显示在主页与分享卡。

✦ 觉醒 & 季节

  • §3 的觉醒形（day30）+ 四季光环（每季一套）+ bond 850/1000 的稀有光环，构成成年后持续的视觉新鲜感。
  • Lv 仍每 2–3 天涨一格、每几级解锁配饰，所以等级始终对应一个具体的解锁，不再是空数字。
🎯 一句话：Lv20 后的玩家，每天上号是为了——推进当前 fork / 凑齐这周的配饰 / 点亮下一颗星 / 看今天的专属心声 / 不断连登拿月度款。升级只是这些的节拍器。
